#### 任职要求:

- 计算机科学或相关领域的学士以上学位;
- 精通 C/C++，和常见各种设计模式;
- 较强的沟通能力和团队合作能力;
- 有强烈的责任心、团队精神，善于沟通合作;
- 对改善开发者感受和产出充满激情，较强的沟通能力和团队合作能力;
